企画　Sleep Tracker

内容
・睡眠時間の計測(寝る前にタイマー押して起きたら止める、また変更)
・睡眠の質はどうだったか(眠い、悪夢、快眠など)
・なにか睡眠に関わる事をしたか(一日の体の疲れ、どれだけｐｃスマホをいじったか、
　コーヒー、直前までゲーム、風呂入ってないなど)
・アラーム、就寝前のリマインド
・睡眠のデータ閲覧、分析


必要画面
・ホーム画面⇒睡眠計測画面(時間計測のbuttonがあるのと、時間変更のbutton)⇒その日の睡眠の質を評価(メモなども)
　　　　　　⇒睡眠カレンダー画面⇒分析など？
　　　　　　⇒アラーム、リマインド画面
　　　　　　⇒睡眠データ画面
